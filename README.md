 Project Overview

The EduChatbot Assistant is an interactive, AI-powered educational tool designed to make complex Artificial Intelligence concepts accessible to beginners. Built entirely on the Botpress
no-code platform, this project demonstrates the power of rapid prototyping and conversational AI for knowledge transfer.

This assistant is specifically focused on providing comprehensive answers, interactive quizzes, and concise summaries related to core AI domains, including 
NLP, Machine Learning, and Deep Learning**.
